Downloads {{include.data_source}} from {{page.toc_title}}. {{ include.data_source }} are accessible as a Pandas Dataframe via the {{ include.data_source }}.df attribute after the download completes.

{{ include.data_source }}.df is overwritten with the new downloaded data. The dataframe is not saved automatically following the download - call {{ include.data_source }}.save() to save the dataset.

In-progress downloads are saved as csv files to local storage.

Arguments

Name Description
created_gt
optional datetime 		Return {{ include.data_source | downcase }} created after this value.
created_gte
optional datetime 		Return {{ include.data_source | downcase }} created on or after this value.
updated_gte
optional datetime 		Return {{ include.data_source | downcase }} updated on or after this avlue.
limit
optional integer 		Limits the number of {{ include.data_source | downcase }} returned.

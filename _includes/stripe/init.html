Initializes a Stripe Invoices dataset, loading the dataset if it was previously saved to storage.

See this example for a full download, update, and save lifecycle implementation in Python.

Arguments

Name Description
base_pickle_filename
optional string 		Specifies the name of the Pickle file the Dataset should be loaded from and saved to. If the file doesn't exit, it will be created when the Dataset is saved.

The default is stripe_invoices.pkl.
